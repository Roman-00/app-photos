# ART Galery

## Pnpm Scripts
### Install dependencies
```
pnpm install
```
#### Compiles and hot-reloads for development
```
pnpm dev
```
или, если нужен доступ по сети
```
pnpm dev --host
```
#### Compiles and minifies for production
```
pnpm build
```
#### Lints and fixes files
```
pnpm lint
```
